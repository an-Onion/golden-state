# Golden State - 实时金价查询微信小程序

这是一个使用 Taro 框架开发的微信小程序，能够实时显示黄金价格信息。

## 技术栈

- **框架**: Taro (React)
- **语言**: JavaScript
- **UI**: 原生微信小程序组件
- **构建工具**: Webpack 5

## 功能特性

- 实时金价显示 (AU9999, AU999, 足金, 国际金价)
- 价格趋势显示
- 一键刷新功能
- 响应式设计

## 项目结构

```
src/
```

## 开发环境搭建

### 1. 安装依赖

```bash
# 安装 Taro CLI
npm install -g @tarojs/cli

# 安装项目依赖 (使用 pnpm)
pnpm install
```

### 2. 启动开发环境

```bash
# 启动微信小程序开发环境
pnpm dev:weapp
```

### 3. 构建项目

```bash
# 构建微信小程序
pnpm build:weapp
```

## 微信小程序申请与上线流程

### 第一步：注册微信小程序账号

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击右上角的"立即注册"
3. 选择"小程序"类型
4. 填写邮箱、密码等基本信息
5. 激活邮箱
6. 选择主体类型（个人/企业）
7. 信息登记（填写真实有效的身份信息）

### 第二步：小程序信息完善

1. 登录小程序管理后台
2. 在"设置"-"基本设置"中完善小程序信息：
   - 小程序名称
   - 小程序头像
   - 小程序介绍
   - 服务类目
   - 网络信息（request合法域名等）

### 第三步：开发配置

1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 使用小程序账号登录开发者工具
3. 获取小程序的 AppID（在"开发"-"开发设置"中查看）
4. 在项目配置文件中添加 AppID

### 第四步：上传代码

1. 在开发者工具中打开项目
2. 点击"上传"按钮
3. 填写版本号和项目备注
4. 上传成功后，在微信公众平台可以看到提交的版本

### 第五步：提交审核

1. 在微信公众平台的"开发管理"-"开发版本"中找到刚上传的版本
2. 点击"提交审核"
3. 填写审核说明，描述小程序的功能和用途
4. 提交后等待微信官方审核（一般1-7个工作日）

### 第六步：发布上线

1. 审核通过后，在"版本管理"中可以看到审核通过的版本
2. 点击"发布"按钮，即可让小程序上线
3. 用户可以在微信中搜索小程序名称找到你的小程序

### 注意事项

- 个人类型小程序无法使用支付功能
- 上传代码前需要配置服务器域名（request合法域名）
- 发布前请充分测试各项功能
- 遵守微信小程序平台规范，避免违规内容

## 本地开发说明

本项目使用 Taro 框架开发，开发流程如下：

1. 修改代码后，开发者工具会自动编译预览
2. 使用 `pnpm dev:weapp` 命令启动开发服务
3. 使用 `pnpm build:weapp` 命令构建生产代码

## API 接入

本项目使用模拟数据展示金价信息。在生产环境中，您需要接入真实的金价 API 服务：

1. 在 `src/services/gold-price-api.js` 中替换 API 地址
2. 修改请求参数和响应格式以匹配您的 API
3. 添加错误处理和重试机制
